Urban — WordPress Theme

Modern WordPress theme for small studios. Full Site Editing (FSE), patterns-first, clean visual foundation.

Description

Urban 3D is a modern WordPress theme designed for small studios, focused on enhancing creative work through a strong, clear, and contemporary visual structure.

It is built on Full Site Editing (FSE), with a pattern-first approach and a solid visual foundation, designed to solve real-world layouts without unnecessary complexity.

Key Features

  • Full Site Editing (FSE) support
  • Pattern-first design approach
  • Custom blocks included
  • Local fonts and optimized assets
  • Clean, dark, modern visual style
  • Fully responsive and mobile-ready

Theme Preview (Layouts & Editor)

What’s Included

  • Block-based templates and template parts
  • Ready-to-use patterns (hero, CTA, team, testimonials, FAQ, contact)
  • Custom blocks:
    • Cover 3D (hero)
    • Header Switch
    • Icons (inline SVG)
    • Slider (Splide bundled)
  • Local font configuration
  • Documentation

Requirements

  • WordPress 5.9 or higher
  • PHP 7.4 or higher
  • Modern browser with JavaScript enabled

Usage Notes

Urban 3D is designed to be edited entirely through the Site Editor. Templates, patterns, and global styles can be customized without additional plugins.

Portfolio functionality is optional and can be added via a separate plugin, keeping content portable and independent from the theme.

Support

This theme is provided as-is. Basic usage guidance is included in the documentation.
